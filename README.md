# vue-gram

A simple instagram clone built with vuejs

# Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run serve
```

### Compiles and minifies for production
